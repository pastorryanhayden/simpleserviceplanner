Built for traditional churches.

There are plenty of tools for large churches with professional teams, lights and magic. This is a tool for churches that still use traditional services: ie. hymns, a human doing announcements, a choir, etc.

Organize songs, hymns, announcements and sermons.

Keep track of the essential elements of upcoming and past services using this easy to use table. Easily access past specials, track which hymns you sing how often, etc.

Quickly generate printed orders of service.

We give you an html file that you can save to your desktop, simply click it to open it in a browser, choose your service from a list, and click print. Save time every week.

Easy access on any computer, tablet or smartphone.

This tool is built on top of Airtable. So you get access via a world class interface on any web browser and awesome apps for your phone.

I built this tool with the church where I serve as pastor in mind. I was sick of thumbing though a hymn book, guessing how long it's been since we sung that song. I wanted my Assistant Pastor, pianists, and musicians to be on the same page.
I sincerely hope that it is a blessing to your church.
